## Instalacion
Pasos para la instalacion del proyecto
```
git clone https://github.com/AubisDev/pruebas-tecnicas.git
```


```
cd pruebas/01-reading-list/aubisdev

pnpm install

pnpm dev
```
For testing
```
pnpm test
```
